# Contributing

1. Clone the repo and install JDK 11+ (JDK 17 for collision map generation).
2. Run `.\gradlew :loader:runDev`.
3. Match existing style (`net.solace.*`, Java 11, focused diffs).
4. Confirm `.\gradlew build` passes before opening a PR.

RuneLite version bumps: see [`mappings/README.md`](mappings/README.md).
